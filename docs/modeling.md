# Modeling

This stage trains baseline models on the processed Kepler vetting dataset.

## Inputs

The first baseline uses the filtered model-ready dataset:

    data/processed/kepler_q1_q17_dr25_model_ready.npz

This file is generated by:

    PYTHONPATH=src python -m kepler_vetting.processing.filter_model_ready_dataset

## Tabular baseline

Train the first boring baseline:

    PYTHONPATH=src python -m kepler_vetting.modeling.train_tabular_baseline

The script trains:

1. `DummyClassifier(strategy="most_frequent")`
2. `LogisticRegression(class_weight="balanced")`

It uses repeated stratified train/validation/test splits across multiple random seeds.

For each seed, the split is:

- 60% train
- 20% validation
- 20% test

To avoid validation/test leakage, tabular features are reconstructed from the stored processed features and then standardized inside each seed using `StandardScaler` fit on the training split only.
