| Field           | Meaning                                       |
| --------------- | --------------------------------------------- |
| `koi_period`    | Orbital period, in days.                      |
| `koi_duration`  | Transit duration, usually in hours.           |
| `koi_depth`     | Transit depth, usually in parts per million.  |
| `koi_prad`      | Estimated planet radius, in Earth radii.      |
| `koi_teq`       | Estimated equilibrium temperature, in Kelvin. |
| `koi_insol`     | Insolation flux relative to Earth.            |
| `koi_model_snr` | Transit model signal-to-noise ratio.          |
| `koi_steff`     | Stellar effective temperature, in Kelvin.     |
| `koi_slogg`     | Stellar surface gravity, log10 cgs.           |
| `koi_srad`      | Stellar radius, in solar radii.               |
| `koi_kepmag`    | Kepler-band magnitude.                        |
